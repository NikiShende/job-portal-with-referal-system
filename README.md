# Job Portal With Referral System

A full-stack Job Portal and Referral System developed to connect job seekers, employees, and recruiters on a single platform. The application allows users to explore job opportunities, apply for jobs, and receive referrals from employees to improve hiring chances.

## 🚀 Features

* User Authentication & Authorization
* Job Posting and Management
* Referral Request System
* Resume Upload Functionality
* Role-Based Access (Admin, Recruiter, User)
* Search and Filter Jobs
* Company and Job Details Page
* Responsive User Interface
* Secure Backend APIs
* Database Integration

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/NikiShende/job-portal-with-referal-system.git
```

### Frontend Setup

```bash
cd job portal frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd job-portal-backend
nodemon index.js
```
